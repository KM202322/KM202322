## 🚀 What I'm Building

### 🤝 MyoMate — Telegram Mini App for Myanmar Youth
Friend-matching & anonymous confessions, built **ethics-first**.
**Stack:** Python (aiogram) · React/Vite · aiohttp · Supabase/PostgreSQL · Render
**Highlights:** Burmese-first UI · anonymous confessions (identity never exposed, even to admins) · private password-protected rooms · need-to-know moderation with audit logs · strict 18+ gate, no cash-out, organic growth over spam.
🔗 [Live App](https://p1-254e.onrender.com)

A refined web presence for a **Mogok, Myanmar** gemstone company.
**Stack:** PHP 8.2 · MySQL · cPanel Git deploy
**Highlights:** Ruby Red / Metallic Gold brand identity · Cinzel/Poppins typography · gem-rain canvas animation · 360° rotating CSS gems · inquiry system & privacy-respecting analytics.
🔗 [rubyrain.kyawwinthet.online](https://rubyrain.kyawwinthet.online)
